== Popis ==
* Umožňuje přidat emailu s informací o zpracovávané objednávce dvě přílohy a ke každému produktu v objednávce po jedné příloze.

== Instalace ==
* Plugin je možné nainstalovat pomocí FTP, nebo nahráním Zip souboru v administraci

= Minimální požadavky =
* WordPress 4.0 nebo vyšší
* PHP version 5.2.4 nebo vyšší
* MySQL version 5.0 nebo vyšší
* WooCommerce 2.1
